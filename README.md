## 设计原则

- **量子跃迁（Quantum Leap）**：不是过渡，是阈值——A 消失，B 出现，无路径，无回头。
- **Threshold Cyan `#00C1D4`** 仅作为阈值线（Edge），横向贯穿，不做装饰高亮。
- **Energy Orange `#FA5F26`** 全站只此一处（主 CTA 的 12px peak node）。
- **Quantum Black `#000000`** 承载 before-state 的密度。
- 所有色值用 OKLCH，所有字号用 `clamp()`，所有间距走 8pt 栅格。
- 动效仅 `transform / opacity / filter`，硬切（cubic-bezier(0.16, 1, 0.3, 1)），无渐变 / 无粒子 / 无 spring。
- 完整支持 `prefers-reduced-motion`（替换为 1ms 而非删除）。
